## Рекомендуемый запуск:  

> [!INFO] Для Unix-подобных ОС (MacOS & Linux)

Используйте компилятор `g++`, чтобы получить сборку написанного кода:
```bash
g++ -std=c++17 путь_к_cpp_файлу -o путь_к_билду
```

Далее запустите билд:  
```bash
путь_к_билду
```
